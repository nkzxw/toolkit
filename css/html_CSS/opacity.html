我是不透明的div

.div{
    overflow:hidden;
    margin:10px 0 0;padding:0 15px;width:600px;height:200px;
    background:#000;
}

使用opacity实现的半透明

.opacity50{ /* chrome,firefox,safari,opera,ie9+ */
    background:#000;
    opacity:0.5;
}
.lte8 .opacity50{ /* ie6-8 */
    filter:alpha(opacity=50);
    zoom:1; /* ie6,ie7下的行内元素必须要出触发haslayout，透明效果才有效 */
}

使用RGBa完美实现半透明

.rgba{ /* chrome,firefox,safari,opera,ie6-8 */
    background:rgb(0,0,0);  /* 声明一个保留颜色 */
    background:rgba(0, 0, 0, 0.5);
    filter:progid:DXImageTransform.Microsoft.Gradient
    (GradientType=0, StartColorStr='#7e000000', EndColorStr='#7e000000');
}
:root .rgba{ /* ie9+ */
    filter:progid:DXImageTransform.Microsoft.Gradient
    (GradientType=0, StartColorStr='#00000000', EndColorStr='#00000000');
}
