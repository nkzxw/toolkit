<script type="text/javascript">
            window.onload = function(){
                var prep = document.getElementsByTagName('p')[0];
                document.onkeydown = function(event){
                    var e = event || window.event;
                    prep.innerHTML = e.keyCode;
                    return false;
                }
            }
        </script>

这里将显示您按下键的keyCode值
